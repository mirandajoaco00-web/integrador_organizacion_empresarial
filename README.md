# Tp-Organizacion-Empresarial
## 📖 Guía Rápida de Uso 

1. **Iniciar el sistema:** Ejecutar el archivo de Python desde la terminal corporativa.
2. **Autenticación (Estado 1):** Ingrese su DNI numérico sin puntos ni espacios (Ej: `1234567`). Para finalizar el programa de forma anticipada, escriba `salir`.
3. **Solicitud de Días (Estado 2):** El bot indicará su saldo actual. Tipee un número entero positivo menor o igual a sus días disponibles en pantalla.
4. **Formato de Fecha (Estado 3):** Introduzca la fecha usando barras diagonales con el formato estricto `dd/mm/aaaa` (Ej: `18/10/2026`).
